# Telco Customer Churn Prediction

## 📌 Project Overview
This project predicts customer churn for a telecom company using Logistic Regression.

## 🔍 Problem Statement
Customer churn impacts company revenue. This model helps identify high-risk customers so retention strategies can be applied.

## 🛠 Steps Performed
- Data Cleaning
- Handling Missing Values
- Feature Encoding
- Feature Scaling
- Class Imbalance Handling
- Model Building (Logistic Regression)
- Model Evaluation using ROC-AUC

## 📊 Model Performance
- Accuracy: 73%
- Recall (Churn): 80%
- ROC-AUC: 0.83

## 💻 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📈 Business Impact
The model helps the company proactively retain customers and reduce revenue loss.
